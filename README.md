# 基于A*搜索的多无人机-无人车汇合路径搜索算法
## 多无人机与无人车汇合
每个智能体并行执行A*搜索，根据各搜索得到的关闭列表选取汇合位置，具体算法见论文。

无人机数量不限，无人车仅包含一辆。

## 模块说明
**调用库**
- `Agent`: 智能体基类
- `UAV`: 无人机类
- `UGV`: 地面车辆类
- `A*`: 核心路径搜索算法，支持 2D/3D 环境
- `NodeMap`: 单例模式的地图管理器，基于体素地图构建
- `Node`: 栅格节点，存储代价值和路径信息

**具体实现**
- `One2One`: 单无人机与单无人车汇合路径规划（并行/串行）
- `Multi2One`: 多无人机与单无人车协同汇合（并行，向下兼容）
- `Standard`: 标准A*路径规划

智能体各自并行调用A*方法类执行搜索，由具体实现决定搜索的结束方式。
